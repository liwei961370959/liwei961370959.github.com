友盟授权微信登录

Veröffentlicht am 2017-07-26

我觉得友盟的页面找个东西太麻烦了。因为友盟的东西太多。所以我感觉在里面找个东西是很麻烦的。如果你遇到什么问题，去搜索问题基本上找很费劲。好了不吐槽了。

我写这个的主要目的是第一次用友盟的微信登录，友盟都给你封装好了。每一步步应该怎么做。我主要是写文档记录为后期再次使用记录方便。

账号准备

微信账号、友盟账号。以及申请的key。友盟下载的Dome

友盟集成工具

首先这个工具很强，先跟你你的需求生成你需要的jar包。

最好将几个文件全部导入你的项目。

libs主要放你的jar包。

但是其他的文件主要是存放的微信分享的相关图片和布局

还能跟你需求生成相关的代码

目前application中的代码还不能自动生成，这个可能是友盟的问题

# 注意 #：

1：在AndroidManifest.xml中下注册下面的回调Activity，
直接复制到Android manifest XML 即可。所以WXEntryActivity的时候。需要在项目中创建 wxapi 的包用于微信的回调。

2：可以用友盟集成工具中会生成直接复制替换，不要忘记manifest中的将activity进行注册。

3：新建登录页面

注册微信回调代码：

{
//友盟
private UMAuthListener umAuthListener = new UMAuthListener() {
    @Override
    public void onStart(SHARE_MEDIA share_media) {
        Toast.makeText(getApplicationContext(), "onStart", Toast.LENGTH_SHORT).show();
    }

    @Override
    public void onComplete(SHARE_MEDIA platform, int action, Map<String, String> data) {
        Toast.makeText(getApplicationContext(), "授权成功", Toast.LENGTH_SHORT).show();
        umShareAPI.getPlatformInfo(LoginActivity.this, platform, new UMAuthListener() {
            @Override
            public void onStart(SHARE_MEDIA share_media) {
                Toast.makeText(getApplicationContext(), "授权成功", Toast.LENGTH_SHORT).show();
            }

            @Override
            public void onComplete(SHARE_MEDIA share_media, int i, Map<String, String> map) {
                Toast.makeText(getApplicationContext(), "map" + map, Toast.LENGTH_SHORT).show();
                Log.e("----", "map:" + map);//登录信息以map的形式存储的。获取其中的信息应用到自己对应的APP进行登录。
                StringBuilder sb = new StringBuilder();
                Set<String> keys = map.keySet();
                for (String key : keys) {
                    sb.append(key + "=" + map.get(key).toString() + "\r\n");
                }
                loginByWeChat(map.get("unionid").toString(), map.get("screen_name").toString(),
                        map.get("iconurl").toString()

                );
            }

            @Override
            public void onError(SHARE_MEDIA share_media, int i, Throwable throwable) {

            }

            @Override
            public void onCancel(SHARE_MEDIA share_media, int i) {

            }
        });
    }


    @Override
    public void onError(SHARE_MEDIA platform, int action, Throwable t) {
        Toast.makeText(getApplicationContext(), "授权失败" + action + "=--" + platform + "--==" + t, Toast.LENGTH_SHORT).show();
    }

    @Override
    public void onCancel(SHARE_MEDIA platform, int action) {
        Toast.makeText(getApplicationContext(), "取消授权", Toast.LENGTH_SHORT).show();
    }

};
}

遇到的问题

1：如果你按操作步骤一步步弄的。接口回调的时候不走任何办法

可能是你不细心的原因造成的。有的人。下载了微信的登录以及分享的Dome。然后主要的把.wxapi.WXEntryActivity复制的有问题。需要把控各个细节自卸检查下。

2：如果遇到授权失败

这里可以打印onError方法中的Trouble。

一般出现的错误方式是微信的错误。可以去微信的开放平台搜索问题。
